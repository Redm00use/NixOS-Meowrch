## 🚀 О проекте

**Meowrch NixOS** — это красивая, минималистичная и оптимизированная конфигурация для NixOS 25.11. Мы перенесли всю эстетику и удобство оригинального Meowrch на мощную декларативную базу NixOS.

### ✨ Особенности
- **Основа:** NixOS 25.11 (Stable) + Flakes + Home Manager.
- **Интерфейс:** Hyprland с анимациями и размытием.
- **Тема:** Catppuccin Mocha (единый стиль для всех приложений).
- **Производительность:** Оптимизировано для AMD GPU (RADV).
- **Удобство:** Автоматический установщик, меню приложений Rofi, уведомления Dunst.
- **Уникальность:** Собственные скрипты и инструменты из оригинального Meowrch.

## 🛠 Установка

Простой и автоматизированный процесс установки:

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Redm00use/NixOS-Meowrch.git
cd NixOS-Meowrch

# 2. Запустите установщик
chmod +x install.sh
./install.sh
```

**Установщик сам:**
- Определит ваше железо.
- Предложит выбрать имя пользователя и хоста.
- Настроит систему и соберет конфигурацию.

## ✨ Особенности
*   **Hyprland + UWSM**: Современное рабочее окружение с максимальной плавностью и стабильностью сессии.
*   **Mewline**: Уникальный статус-бар в стиле Dynamic Island.
*   **Декларативность**: Ваша система всегда будет одинаковой, на любом железе.
*   **Безопасность**: Откаты к предыдущим версиям прямо в загрузчике.

## 📁 Структура проекта
*   `hosts/meowrch/` — **Здесь живет ваша система**. Основные файлы: `configuration.nix` (система) и `home.nix` (пользователь).
*   `modules/` — «Подкапотные» настройки (драйверы, темы, сервисы).
*   `config/` — Конфиги программ (те самые файлы, которые вы обычно ищете в `~/.config`).
*   `scripts/` — Набор полезных скриптов для управления всем подряд.

## 🔧 Использование и Обновление

Мы создали супер-команды (алиасы), чтобы вам не нужно было учить сложные консольные заклинания:

| Команда | Что делает? |
|---------|-------------|
| `u` (или `г`) | **Полный апдейт**. Скачает обновления с GitHub и пересоберет систему. |
| `b` (или `и`) | **Применить настройки**. Запускайте это каждый раз, когда изменили что-то в конфигах. |
| `clean` | **Уборка**. Удаляет старые версии системы и освобождает место на диске. |

## 📦 Гайд для новичка: Установка программ

В NixOS пакеты записываются в списки. Есть два главных списка:

### 1. Программы для вас (Браузеры, Плееры, Игры)
Откройте файл `hosts/meowrch/home.nix` в редакторе (команда `zed ~/NixOS-Meowrch/hosts/meowrch/home.nix`).
Найдите раздел `home.packages` и добавьте название программы:
```nix
home.packages = with pkgs; [
  telegram-desktop
  vlc
  discord
];
```

### 2. Системные штуки (Драйверы, Низкоуровневые утилиты)
Откройте `modules/nixos/packages/packages.nix`. Добавьте пакет в `environment.systemPackages`.

**Как найти название пакета?**
Зайдите на **[search.nixos.org](https://search.nixos.org/packages)** и введите название. Используйте только название из колонки "Attribute name" (например, `firefox`, а не `Firefox Browser`).

**Как применить?**
После сохранения файла просто напишите в терминале:
`b` (или `и`)

## 📖 Meowrch Wiki: Тонкая настройка

Система Meowrch очень гибкая. Вот где менять «внешку»:

### 1. Горячие клавиши (Хоткеи)
Файл: `modules/home/hypr-configs/keybindings.conf`
Хотите поменять `Super+D` на что-то другое? Вам сюда.

### 2. Автозапуск программ
Файл: `modules/home/hypr-configs/autostart.conf`
Добавьте сюда строку вида `exec-once = название_программы`.

### 3. Обои и Темы
Используйте встроенные меню:
*   `Super + W` — Смена обоев.
*   `Super + Shift + T` — Смена цветовой схемы.

## 🛠️ Что делать, если всё сломалось?

Это NixOS, здесь **невозможно** окончательно «убить» систему программно!

1.  **Если система не загружается**: Перезагрузитесь. В меню загрузки (GRUB/Systemd-boot) выберите предыдущий пункт (Generation). Вы окажетесь в системе, которая работала до ваших правок.
2.  **Если вы в системе, но всё глючит**: Введите команду `rollback` (или `sudo nixos-rebuild switch --rollback`). Система вернется к прошлому рабочему состоянию.

|---------|----------|

| `Super + Z` | Браузер |

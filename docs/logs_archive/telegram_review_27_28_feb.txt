Привет всем! 👋 Мы хорошенько поработали в эти дни и выкатили пачку важных исправлений и улучшений. Вот что мы починили и добавили:

• **Оптимизация:** Очищены логи загрузки (удален лишний таймер обслуживания).
• **Home Manager:** Модуляризирована структура, синхронизированы скрипты в `bin`.

🚧 **Важно:** Тема **SDDM** всё ещё находится в активной разработке! Мы переносим её на Qt6 и чиним конфигурацию.

🚀 **NixOS-Meowrch Update: Feb 27-28 Patches!** 🚀

Hey everyone! 👋 We've been working hard over the last couple of days to bring you a solid batch of fixes and improvements. Here is what we've fixed and added:

🎨 **Theming & UI:**
• Completely overhauled theming logic! Fixed issues with the white theme in Nemo file manager and GTK generation.
• Prioritized system themes and forced dark mode preference. Removed hardcoded cursors and icons from Hyprland config.
• Our tool `pawlette` now has `gsettings` access for proper theme switching and uses absolute Nix Store paths for reliability.
• Wallpapers are now automatically linked from the themes package to the user config.

🐚 **Shell & Terminal:**
• **Fish:** Fixed syntax errors, resolved configuration conflicts, and fixed `fastfetch` startup.
• **Starship:** Eliminated the 'unknown key' error by removing unsupported modules.

💻 **System & Components:**
• **Mewline:** Added brightness detection, laptop/desktop support. Fixed notifications, volume keybindings, and window rules. Enabled `power-profiles-daemon`.
• **Screen Lock (Swaylock):** Implemented meowrch theme effects, corrected lock script paths in Hyprland config.
• **Clipboard (Rofi):** More robust clipboard handling to prevent data loss; removed incompatible rofi flags.
• **Networking:** Fixed false metered connection status in NetworkManager.
• **Optimization:** Cleaned up boot logs by removing a redundant maintenance timer.
• **Home Manager:** Modularized the structure and properly synced the `bin` scripts directory.

🚧 **Note:** The **SDDM theme** is still actively in development! We are porting it to Qt6 and polishing the configuration.
